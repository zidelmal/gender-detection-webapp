About Gender Detector

  1. How does it work?
    • "Gender Detector" uses a Machine Learning algorithm called "ExtraTreeClassifier" to read a name and output whether the name is a boy's name or a girl's name.
    • The algorithm has learnt how to make these output predictions by learning from over 120,000 first names from all over the world(even names writen in arabic).
  2. Does it work for all the names?
    • The algorithm was developed by feeding it with names writen in both latin and arabic alphabets. So it can detect all names written with these alphabets.
  3. Does the website store any information?
    • No. The algorithm runs on the names your provide and plots out the output predictions. The names are lost after that and the app does not store any information.
